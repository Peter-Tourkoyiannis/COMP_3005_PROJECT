TO RUN PROJECT:

1. Have pgAdmin4 installed and running on your PC.
2. Create a database called Health_And_Fitness_Club_Management_System.
3. Run the DDL to create the tables, then the DML to populate the tables.
4. Either Run the test cases or the python application located in /app from the root directory in the git.
